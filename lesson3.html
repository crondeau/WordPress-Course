Planning Templates

Your design will dictate what template your need.

Understanding how templates work is crucial.

For example, a 5 page one column website
with no blog may need:

  • index.php
  • style.css
  • header.php
  • footer.php
  • functions.php

Global files

Some files appear on all of your website pages

  • header.php - displays all the content in your header area
  • footer.php - displays all the cotent in the footer area
  • sidebar.php - displays sidebar content

Index.php

The index file controls what the homepage of your site looks like. By default it is a loop that displays the most recent blog posts, with a link in the bottom to view older posts.

Alternately, you can set a static front page as the
home page by going to:
wp-admin -> settings -> reading

Page.php

Page.php controls what pages look like. Pages are used for static content, such as contact us, about us, etc...

Single.php

The display of individual posts is controlled by the single.php. It contains a loop that queries just one post and displays it.

Archive.php

You can control the look and feel of different archives using this template file. The archive.php will be used when displaying post per category, tag, date, author, etc...

To set up your project

  1. Make a copy of your starter theme in a folder & give it a unique name
  2. Open the stylesheet and edit the comment at the very top
  3. Create a new screenshot(screenshot.png) based on your design
  4. Replace the favicon with a new one
  5. Upload and activate your theme

Exercise

  1. Open and study the PSD provided and determine what templates are needed.
  2. Set up your projects folder with a copy of the Bluelime Basic starter theme.
  3. Create the pages and a few blog posts to help you with the site build.
  4. Set up your home page as the front page and select your blog page to display your posts.
  5. Convert the PSD into HTML and CSS
